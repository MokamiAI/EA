TROUBLESHOOTING:

Q: Build fails with "Command not found: cmake"
A: Install CMake (Windows: MinGW, macOS: brew install cmake, Linux: apt install cmake)

Q: Tests won't run
A: Rebuild entire project: Delete Build/ folder, then Ctrl+Shift+B → Build

Q: IntelliSense not working
A: Ctrl+Shift+P → "C/C++: Reset IntelliSense" (wait for re-indexing)

Q: Can't find include files
A: Check CMakeLists.txt has correct paths, verify file names (case-sensitive)
